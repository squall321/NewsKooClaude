# 국제 유머 번역 플랫폼 개발 로드맵

## 프로젝트 개요
해외(Reddit 등) 유머 콘텐츠를 자동 크롤링하여 한국어로 번역하고, 검수 후 서비스하는 플랫폼

### 기술 스택
- **프론트엔드**: React + TypeScript + Vite
- **백엔드**: Flask (Python)
- **데이터베이스**: PostgreSQL
- **번역**: OpenAI GPT / DeepL API
- **크롤링**: PRAW (Reddit), Beautiful Soup
- **배포**: Docker, Nginx
- **모바일**: React Native (Phase 2)
